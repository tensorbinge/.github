## About

TensorBinge is an open research collective focused on rigorous, reproducible work across three domains:

- **Machine Learning** — deep learning theory, optimization, representation learning, and neural architectures
- **Quantitative Finance** — systematic strategy research, statistical arbitrage, risk modelling, and market microstructure
- **LLM Engineering** — applied language model research, fine-tuning, inference, and evaluation

We work in public. Experiments ship as repositories. Notes become papers when they earn it.

## Getting Involved

- Browse repositories and open issues marked `good first issue` or `research`
- Discussions for questions, proposals, and collaboration are open on each repo
- Contributions follow standard fork → branch → PR workflow
- For research proposals or collaboration inquiries, open a Discussion

---

## Principles

1. **Reproducibility first** — every result ships with the code and config that produced it
2. **Clarity over cleverness** — readable implementations over optimised-but-opaque ones
3. **Incremental publication** — share intermediate findings early, iterate in public
4. **Rigour without gatekeeping** — high standards, low barriers to entry
