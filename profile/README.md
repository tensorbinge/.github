## About

Tensorbinge builds open source tools and runs experiments in machine
learning, quantitative finance, and systems engineering.

## Research Areas

| Domain           | Topics                                                   |
| ---------------- | -------------------------------------------------------- |
| ML Research      | optimization, generalization, architectures, theory      |
| Quant Finance    | alpha research, factor models, execution, microstructure, backtesting |
| LLM Applications | prompting, RLHF, RAG, evaluation, fine-tuning            |
| Open Experiments | benchmarks, datasets, tooling, reproductions, logging    |
